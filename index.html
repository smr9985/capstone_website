Scoring Tracking Association using MLPs

Overview

This page contains information for a capstone project, done as part of the requirements for the Computer Science MS program at RIT.

The goal of this project is to explore multi-object radar tracking by using a Multilayer Perceptron, or MLP, to score track-to-detection associations. The learned association score will eventually be compared against classical distance-based association methods.

General Background

Multi-Object Tracking

Multi-object tracking systems estimate the motion of multiple targets over time. A common tracking pipeline follows a predict, associate, and update cycle. In this process, new detections are matched with existing tracks in order to maintain target identity across frames.

FMCW Radar

This project focuses on simulated FMCW radar data. FMCW radar is useful for short to medium range tracking because it can provide radar-related measurements such as range, radial velocity, and return power.

MLP-Based Association

The proposed approach is to use an MLP to score track-detection pairs. Instead of relying only on traditional distance-based metrics, the model can learn from features such as position, range, velocity, and power.
